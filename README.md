# demo-management-system
A Java-based Student Management System designed with a clean Swing interface. It handles  generating student IDs and storing records safely in text files. You can easily add, view, and manage BITS student details, and there’s even a built‑in tool to instantly generate batches of CS students. 
## Features

- Add new student records  
- View all students in a scrollable list  
- Automatically generate unique IDs in the specific format followed by BPDC
- Generate 100 CS students for testing or demonstration  
- Save and load student data using a text file  
- Simple and intuitive Swing GUI  
- Clean object‑oriented code structure  

---

## Tech Stack

- Java (JDK 8 or higher)  
- Swing for GUI  
- File I/O for data storage  
- VS Code as the development environment
